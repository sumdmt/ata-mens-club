# ATA MEN’S CLUB - Kuaför Randevu Sistemi

Modern ve kullanıcı dostu web tabanlı kuaför randevu sistemi.

## Kullanılan Teknolojiler

### Frontend

* React.js
* CSS3
* React Icons

### Backend

* Node.js
* Express.js
* JWT Authentication

### Veritabanı

* MongoDB Atlas
* Mongoose

---

# Özellikler

## Müşteri Tarafı

* Randevu oluşturma
* Çalışan seçme
* Birden fazla işlem seçebilme
* Otomatik süre hesaplama
* Dolu saat kontrolü
* Kapalı saat kontrolü
* Pazar günü randevu engeli
* 22:00 sonrası taşan randevu engeli

---

## Admin Paneli

* Admin giriş sistemi
* JWT token koruması
* Randevu onaylama
* Randevu reddetme
* Randevu düzenleme
* Randevu silme
* Tarihe göre filtreleme
* Çalışana göre filtreleme
* Günlük / gelecek / geçmiş randevu sekmeleri

---

## Kapalı Saat Sistemi

* Çalışan bazlı izin oluşturma
* Başlangıç ve bitiş saati belirleme
* Açıklama ekleme
* Geçmiş izinlerin otomatik silinmesi

---

## Raporlama Sistemi

* Toplam randevu sayısı
* Bekleyen randevular
* Onaylanan randevular
* Reddedilen randevular
* Günlük gelir
* Aylık gelir
* Yıllık gelir
* Çalışan bazlı gelir raporu
* En çok yapılan işlemler

---

# Güvenlik

* JWT doğrulama
* Helmet güvenlik middleware
* Express Rate Limit
* CORS koruması
* Environment variable kullanımı

---

# Çalışma Saatleri

* Pazartesi - Cumartesi
* 08:00 - 22:00
* Pazar kapalı
